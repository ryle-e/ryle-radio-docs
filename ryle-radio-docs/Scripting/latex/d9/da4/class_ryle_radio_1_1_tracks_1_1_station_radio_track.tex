float Ryle\+Radio.\+Tracks.\+Station\+Radio\+Track.\+threshold\+Before\+Repeats}



When \doxylink{class_ryle_radio_1_1_tracks_1_1_station_radio_track_a9c325018c6641f7f483127889959ccd4}{random\+Sequence} is true, this is the number of other tracks that need to be played before the same one is chosen again. Stops the same track from playing back-\/to-\/back, and forces variety in the track order. 

The number of tracks to be played before one can be played again is {\ttfamily round\+\_\+down( (track\+\_\+count -\/ 1) \texorpdfstring{$\ast$}{*} threshold )}. ~\newline
i.\+e with four tracks and a threshold of 0.\+5f, {\ttfamily round\+\_\+down((4 -\/ 1 == 3) \texorpdfstring{$\ast$}{*} 0.\+5) == 1}\+: one other track will need to be played before a repeat ~\newline
i.\+e with four tracks and a threshold of 0.\+7f, {\ttfamily round\+\_\+down((4 -\/ 1 == 3) \texorpdfstring{$\ast$}{*} 0.\+7) == 2}\+: two other tracks will need to be played before a repeat ~\newline
i.\+e with four tracks and a threshold of 1f, {\ttfamily round\+\_\+down((4 -\/ 1 == 3) \texorpdfstring{$\ast$}{*} 1) == 3}\+: three other tracks (all other tracks) will need to be played before a repeat ~\newline
i.\+e with eleven tracks and a threshold of 0.\+8f, {\ttfamily round\+\_\+down((11 -\/ 1 == 10) \texorpdfstring{$\ast$}{*} 0.\+8f == 8}\+: eight other tracks will need to be played before a repeat {\itshape Do note that if this is set to 1, the tracks are forced to play in the same randomized sequence repeatedly}
