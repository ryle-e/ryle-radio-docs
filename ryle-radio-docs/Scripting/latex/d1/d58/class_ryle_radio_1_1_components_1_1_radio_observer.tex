\doxysubsection{Detailed Description}
A component used to watch for specific happenings on a \doxylink{class_ryle_radio_1_1_components_1_1_radio_output}{Radio\+Output}, e.\+g\+: a clip being a certain volume, a track starting, the tune being in a certain range~\newline
We don\textquotesingle{}t inherit from Radio\+Component here since we don\textquotesingle{}t need to use a \doxylink{class_ryle_radio_1_1_radio_data}{Radio\+Data} ref, but it\textquotesingle{}s very similar. 

One major thing to notice about this class is that each individual Observer component has specific tracks it\textquotesingle{}s watching for-\/ NOT each individual \doxylink{class_ryle_radio_1_1_components_1_1_radio_observer_1_1_observer_event}{Observer\+Event}. This is due to limitations of Multiselect\+Attribute-\/ it doesn\textquotesingle{}t display in nested lists properly. This is theoretically something I can fix, but I\textquotesingle{}m not fantastic at custom editors so I\textquotesingle{}d prefer to accept this limitation for now. With that being said, this means it {\itshape is} a bit cleaner to navigate multiple Observers in the scene-\/ not all bad!
